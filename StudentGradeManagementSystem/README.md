# Student Grade Management System

## 📌 Project Overview

The Student Grade Management System is a console-based Core Java application developed to manage student academic records efficiently. The system allows users to add, search, display, delete students, and identify the topper based on average marks.

## 🚀 Features

* Add Student Details
* Display All Students
* Search Student by ID
* Delete Student Records
* Show Topper Details
* Calculate Total Marks and Grade Automatically
* File Handling for Data Persistence
* Exception Handling for Invalid Inputs
* Menu-Driven Console Interface

## 🛠️ Technologies Used

* Java

* Collections Framework (ArrayList)

* Exception Handling
* OOP Concepts

1. Clone the repository.
2. Open the project in Eclipse IDE.
3. Run `Main.java` as a Java Application.
4. Use the menu options to manage student records.
